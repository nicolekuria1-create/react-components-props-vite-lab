# Personal Blog (React + Vite)

This project is a static personal blog built to practice React component design and prop passing.

## Component tree

- App
- Header
- About
- ArticleList
- Article

App renders Header, About, and ArticleList as direct children.
ArticleList renders a collection of Article components.

## Run the project

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the local URL shown in the terminal (typically http://localhost:5173).

## Run tests

```bash
npm test
```

## Build for production

```bash
npm run build
```
